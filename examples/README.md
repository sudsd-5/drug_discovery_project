# Examples

This directory contains example scripts and notebooks demonstrating how to use the Drug Discovery Platform.

## Available Examples

### 1. basic_usage.ipynb
**Description:** Introduction to basic functionality
**Topics Covered:**
- Molecular feature extraction
- Protein sequence analysis
- Loading pre-trained models
- Making predictions
- Visualizing results

**How to run:**
```bash
jupyter notebook examples/basic_usage.ipynb
```

## Running Examples

### Prerequisites

Make sure you have installed all dependencies:
```bash
pip install -r requirements.txt
```

### Jupyter Notebooks

1. Start Jupyter:
   ```bash
   jupyter notebook
   ```

2. Navigate to the `examples/` directory

3. Open the desired notebook

### Python Scripts

Run Python examples directly:
```bash
python examples/example_script.py
```

## Example Data

Some examples use sample data. To download example datasets:

```bash
# Download sample data (when available)
python scripts/download_sample_data.py
```
